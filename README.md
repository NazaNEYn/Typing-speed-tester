# Typing Speed Tester

## Description
This is a simple Python program that measures your typing speed in Words Per Minute (WPM) and calculates typing accuracy. The program provides a GUI where you can type a given reference text, and it tracks the time taken while you type.

## Features
* Simple and clean Tkinter GUI
* Live timer updates while typing
* Calculates WPM based on typed words and time elapsed
* Calculates accuracy by comparing typed words to the reference text
* Start and Stop buttons to control the test

## Modules Used
* `tkinter`: Used to create the graphical user interface, including buttons, text boxes, and labels.
* `time`: Used to track the start and stop times, and calculate elapsed time for typing speed.
